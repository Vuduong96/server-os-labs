## Key Concepts Recap

- DHCP automates IP address assignment using the **DORA process** (Discover, Offer, Request, Acknowledge)
- The main config file is `/etc/dhcp/dhcpd.conf`
- Lease records are stored in `/var/lib/dhcp/dhcpd.leases`
- DHCP listens on **UDP port 67**
